Core Members

Core Members are members registered before June 01, 2009.

Core Members have privileges as compared to users with ended membership, but Core Members access is limited in the following way:

  • Number of achievements is limited to 10
  • Odds games are not accessible
  • Clubs and Schools are not accessible
  • Number of games per day is limited to:
    Type of Game Number per Day
    Bullet30
    Blitz20
    Standard10
    Crazy House10
    Fischer Random10
    Losers10
